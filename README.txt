1. Project Description
----------------------
This is a console-based Library Management System written in C++.
It uses Object-Oriented Programming (OOP) principles including:
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism

Features:
- Add Books and Magazines
- Search by ID
- Borrow and Return Items
- Display All Items
- File storage for data persistence
